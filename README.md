PowerShell Tools for SSRS
=========
PowerShell cmdlets to make interacting with SQL Server Reporting Services simpler.

Cmdlets
=========
Invoke-SrsReport
---------
```
Invoke-SrsReport "http://myserver/ReportServer" "MyFolder/MyReport"
```
